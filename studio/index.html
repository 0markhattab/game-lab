How this works

01. Read the concept

Each lesson opens with the why. What problem are we solving, and why does it matter for game design?

02. Edit the live code

Every lesson has a real working example you can modify. Change a number, hit run, see what breaks. Cmd/Ctrl+Enter runs.

03. Beat the challenge

End-of-lesson challenges are auto-graded. Pass them all, then remix the game templates into something of your own.
