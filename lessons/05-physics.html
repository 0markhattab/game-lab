Velocity & Gravity

Up till now you've moved things with x += 5. That's a constant. Real-feeling motion comes from changing how fast something moves. Today we add the missing layer: velocity, acceleration, and gravity — the soul of every platformer.

05.1

Three layers of motion

Every moving object in physics has three values:

// position — where it is
let x = 100, y = 100;

// velocity — how fast it's moving (and which direction)
let vx = 0, vy = 0;

// acceleration — how the velocity is changing (e.g. gravity)
const gravity = 0.5;

Each frame, you update them in the right order:

function update() {
  vy += gravity;   // acceleration changes velocity
  x  += vx;        // velocity changes position
  y  += vy;
}

That's it. Three lines. From those three lines you get jumping, falling, throwing, bouncing — every movement that feels physical.

05.3

Tuning is the game

Game feel lives in three numbers: gravity, jumpStrength, and the player's horizontal speed. Try these:

Floaty

gravity = 0.3
jump = -10

Like swimming. Long airtime. Fits dreamy or space games.

Snappy

gravity = 1.0
jump = -16

Sharp, responsive. Geometry Dash and Celeste live here.

Heavy

gravity = 1.5
jump = -10

Tank-like. The player feels weight and commitment to every leap.
